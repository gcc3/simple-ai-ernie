
Simple AI Ernie
===============


Node for Ernie Bot.


Setup
-----

`npm i`  


Run
---

`node generate.js`  

It provide 2 APIs:

1. ping  
`/`  

2. generate  
`/api/generate?input=query_string&histories=history_json_string&files=files_text_json_string`

* Follow the node format defined in [simple-ai-chat](https://github.com/gcc3/simple-ai-chat)
